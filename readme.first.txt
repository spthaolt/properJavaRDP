Terminal Service client plugin for ASA

This package contains the source used to build the Terminal Service
client plugin for Cisco ASA clientless SSL VPN rdp-plugin.090915.jar
released on 14 Oct 2009. It contains open source package 
properJavaRDP, see rdp\properJavaRDP-1.1\readme.txt for details.

This package can be built with with JDK 1.5.0_10 and apache-ant-1.7.1
